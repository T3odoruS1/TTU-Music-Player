Steps needed to get this code up and running.
1)Install pygame and tkinter
2)Make sure the folder with music is called "MusicFolder"
3)Images should be in "img" folder
4)Import some music in music folder

1)Установить pygame, tkinter
2)Убедиться, что папка с музыкой называктся "MusicFolder"
3)Все картинки, нужные для программы содержутся в папке "img"
